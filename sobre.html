Objetivo Geral

Organizar e disponibilizar via sistema web uma base de dados climáticos das regiões do MATOPIBA e da região do cinturão citrícola do estado de São Paulo, bem como analisá-la para que possa auxiliar estudos de manejo de recursos hídricos, para atender os objetivos de segurança alimentar, utilizando métodos de Aprendizado de Máquina.
